# PYPOLARON

This repository contains scripts for automated polaron calculations with FHI-AIMS.
 
## Installation

Run:

`pip install -e .`

to install the package via `pip` in editable mode. 

Run the command:

`pip install -e .[dev]`

to install the package and tests used for development.
